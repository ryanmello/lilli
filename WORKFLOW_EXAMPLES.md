## Key Workflow Patterns

### 1. **Single Agent Pattern**
- Simple queries â†’ One agent â†’ Quick response
- Examples: stock checks, basic info
- Fastest execution (~2 seconds)

### 2. **Sequential Pattern**
- Complex queries requiring ordered steps
- Each agent builds on previous agent's output
- Examples: Order creation with design + inventory + pricing
- Moderate speed (~4-5 seconds)

### 3. **Parallel Pattern**
- Independent queries that can run simultaneously
- Significant time savings
- Examples: "What's popular AND what's in stock"
- Optimized speed (~2.5 seconds vs 4 sequential)

### 4. **Error Recovery Pattern**
- Detect issues early
- Provide alternatives
- Maintain positive UX
- Examples: Out of stock, invalid orders

### 5. **Iterative Pattern**
- Agent calls tools multiple times
- Refines query based on results
- Examples: Finding alternatives, optimization
- Variable time based on iterations
